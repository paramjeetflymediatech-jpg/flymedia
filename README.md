# Flymedia Project Management System

This repository contains the source code for the Flymedia Project Management System.

## Structure

- **backend**: Node.js/Express API with MongoDB and AWS S3 storage.
- **web**: Next.js (App Router) SaaS-ready web application.
- **mobile**: React Native (Expo) Android application.

## Getting Started

Please refer to the README files in each subdirectory for specific setup instructions.
